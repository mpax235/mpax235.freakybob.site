Get a PikiDiary User's public info
This tool can get a PikiDiary's User's public info using 5quirre1's PikiDiary API Fan-Made Remake! Emotes support is still being worked on.
This tool can retrieve a User's Profile Picture, Bio, Posts, Achievements, Badges, Banner, Background, Following Count, Followers Count, User ID, Username, Status (Active, Inactive), Login Streak, and Cursor.
Made with HTML, CSS, and JavaScript.
