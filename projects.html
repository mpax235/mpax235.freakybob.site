Android All-in-One Script
This project can reboot you to the OS, Fastboot, Fastbootd, Recovery, FEL Mode, Odin Mode, Meta Mode, Advanced Meta Mode, Factory Mode, Preloader Mode, and Download Mode.
It can dump the BootROM (MediaTek only), or unlock and relock the bootloader using the official way or a different way.
It can also dump the Android Framework, the bootanimation.zip archive if it exists, and the bootaudio.mp3 audio file if it exists.
